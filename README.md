# Project-Orion
An Overengineered Keyboard

<img width="1559" height="567" alt="image" src="https://github.com/user-attachments/assets/21cf839f-5e33-4159-bafd-45fe3830e057" />

This is the repo for a keyboard I designed with Hack Club!

## Features:
 - 2.4ghz / Bluetooth / Wired connectivity, with a custom dongle for 2.4Ghz!
 - Per-key RGB Lighting (Can consume 3.5A at full brightness ðŸ˜­)
 - Low Profile Design (Hotswappable Kailh Choc v1)
 - Reverse-Charging Devices from its own battery
 - OLED Screen to display animations / A GUI :D
 - 3A Fast Charging


## Why I made this:
 - I like engineering and enjoy the challenge of learning how to do something like this from scratch.
 - I like mechanical keyboards, and making my own custom keyboard would be sick.
 - Most keyboards feel like they're trending towards uniformity, and I don't like that.
